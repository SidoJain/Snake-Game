## Getting Started

### Installation

1. Clone the repository:

```bash
git clone [https://github.com/SidoJain/Snake-Game.git](https://github.com/SidoJain/Snake-Game.git)
cd Snake-Game
```

2. Install dependencies using pip:

```bash
pip install -r requirements.txt
```

### Running the Game

Run the main game script:

```bash
python main.py
```
